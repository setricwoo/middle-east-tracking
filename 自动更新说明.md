# 实时新闻自动更新说明

## 功能说明
每20分钟自动执行财联社新闻爬虫，更新 `news.html` 网页，并自动推送到GitHub。

## 文件说明

| 文件 | 说明 |
|------|------|
| `auto_update_news.py` | 主更新脚本（执行爬虫+Git提交+推送） |
| `scrape_cls_final.py` | 财联社新闻爬虫脚本 |
| `run_20min_loop.bat` | 循环执行脚本（保持窗口运行，每20分钟执行） |
| `setup_20min_task.bat` | Windows任务计划设置脚本 |
| `auto_update_log.txt` | 更新日志文件 |

## 使用方法

### 方法一：循环执行（推荐）
双击运行 `run_hourly_loop.bat`，保持窗口打开即可每20分钟自动更新。

```bash
run_hourly_loop.bat
```

### 方法二：Windows任务计划程序
1. 按 `Win + R`，输入 `taskschd.msc` 打开任务计划程序
2. 点击右侧"创建基本任务"
3. 名称：`MiddleEast_News_AutoUpdate`
4. 触发器：每天，每隔1小时重复一次
5. 操作：启动程序
   - 程序：`python`
   - 参数：`D:\python_code\海湾以来-最新\auto_update_news.py`
6. 完成

### 方法三：手动执行
```bash
python auto_update_news.py
```

## 脚本特性

1. **智能去重**：不会重复添加已存在的新闻
2. **增量更新**：新新闻自动添加到页面最上方
3. **高频更新**：每20分钟检查一次新新闻
4. **失败重试**：GitHub上传失败会自动20分钟后重试，最多3次
5. **日志记录**：所有操作记录在 `auto_update_log.txt`

## 查看日志

```bash
type auto_update_log.txt
```

## 停止自动更新

- **方法一**：关闭 `run_20min_loop.bat` 窗口
- **方法二**：删除Windows任务计划中的 `MiddleEast_News_AutoUpdate` 任务

## 注意事项

1. 确保网络连接正常
2. GitHub仓库需要已配置好SSH密钥或已登录凭证
3. 财联社网站结构变更可能导致爬虫失效，需及时更新
