# 02-美以伊冲突每日简报 (briefing.html)

## 页面目标
创建一个专业的每日简报页面，展示美以伊冲突的最新战况、时间线、市场影响和投行观点，帮助用户快速了解冲突进展。

## 页面标题
【华泰固收】中东地缘跟踪 - 美以伊冲突每日简报

## 布局结构

### 整体布局
- **最大宽度**：900px，居中显示
- **背景**：#f8fafc（浅灰蓝）
- **内边距**：24px 20px

### 内容区域（从上到下）

#### 1. 简报头部（高亮框）
渐变黄色背景框（#fef3c7 → #fde68a）：
- **标题**：美以伊冲突第X天简报（日期）
- **副标题**：霍尔木兹海峡封锁第X天
- **核心摘要**：3-4句话概括当日最重要进展

#### 2. 关键数据卡片（4列网格）
显示实时计算的关键指标：
- 冲突天数
- 封锁天数
- 布伦特原油价格
- 油价变化幅度

#### 3. 时间线区块（两栏布局）
**左栏：美以方动态（蓝色主题）**
- 时间线形式展示美国、以色列的军事行动
- 每个节点：时间 + 事件描述

**右栏：伊朗方动态（红色主题）**
- 时间线形式展示伊朗的军事回应
- 每个节点：时间 + 事件描述

#### 4. 市场影响分析
几个高亮信息框：
- **油价影响**（蓝色边框）：当前价格、预测、关键价位
- **航运影响**（橙色边框）：海峡通行量、保险费率、滞留船只
- **产量影响**（红色边框）：各国减产数据、供应缺口

#### 5. 投行观点汇总
一个展开式区域，汇总各大投行最新观点：
- 高盛（日期）：油价目标、配置建议、风险提示
- 摩根士丹利（日期）：油价中枢、全球通胀预测
- 摩根大通（日期）：基准情景、交易建议
- 花旗（日期）：目标价、多空策略
- 美银（日期）：滞胀风险、资产配置

#### 6. 深度分析文章
2-3个重点专题：
- 标题 + 摘要（2-3段）
- 支持数据/图表

#### 7. 相关新闻链接
底部卡片式布局：
- 每条新闻：标题 + 来源 + 摘要
- 链接到财联社原文


## 数据来源

- **战况时间线**：AI自动生成（update_briefing_grok.py）
- **市场数据**：实时获取API
- **投行观点**：手工整理或AI汇总
- **更新频率**：每12小时（北京时间08:00和20:00）

## AI生成提示词（供参考）

当使用AI生成简报内容时，使用以下提示词结构：

```
请生成美以伊冲突第X天的每日简报，包含：
1. 双方最新军事行动（时间倒序，最近24小时）
2. 关键人物/设施损失情况
3. 油价、航运、产量数据变化
4. 投行观点汇总
5. 未来24-48小时关键关注点

要求：
- 专业金融分析师风格
- 数据驱动，引用具体数字
- 客观中立，区分事实与推测
- 中文输出
```
