# 中东地缘政治跟踪系统 - Prompt使用指南

## 文件结构

```
prompts/
├── README.md              # 本文件
├── 00-MASTER_PROMPT.md    # 主控Prompt - 项目整体规范
├── 01-AGENT-地图.md        # 地图Agent - 海湾原油图谱
├── 02-AGENT-简报.md        # 简报Agent - 每日简报生成
├── 03-AGENT-跟踪.md        # 跟踪Agent - 海峡航运数据
└── 04-AGENT-新闻.md        # 新闻Agent - 实时新闻爬虫
```

---

## 使用流程

### 1. 首次阅读

请先阅读 **00-MASTER_PROMPT.md**，了解项目的整体架构、设计规范和各Agent的分工。

### 2. 选择对应Agent

根据你的任务选择对应的Agent手册：

| 如果你要... | 阅读 |
|-----------|------|
| 创建或修改地图页面 | 01-AGENT-地图.md |
| 生成每日简报 | 02-AGENT-简报.md |
| 更新航运数据 | 03-AGENT-跟踪.md |
| 开发新闻爬虫 | 04-AGENT-新闻.md |

### 3. 跨Agent协作

当修改涉及多个页面时（如导航栏调整），请先查阅主控Prompt中的协作规则，确保所有页面的改动保持一致。

---

## 各Agent核心职责

### 地图Agent
- 构建Leaflet交互式地图
- 维护国家、港口、管道三层数据
- 更新设施受冲突影响的状态
- 确保移动端地图可用

### 简报Agent
- 构建AI提示词获取深度分析
- 解析JSON数据生成HTML
- 确保分析内容字数和质量达标
- 保持金融市场数据的准确性

### 跟踪Agent
- 每日更新CSV历史数据
- 维护Chart.js趋势图表
- 更新统计仪表板数字
- 整理最新情报摘要

### 新闻Agent
- 运行Playwright爬虫
- 自动分类新闻内容
- 实现筛选功能
- 保持新闻的时效性

---

## 重要规范

**时间基准**
- 冲突开始：2026年2月28日
- 海峡封锁：2026年3月2日
- 所有页面时间戳必须一致

**数据来源**
- 航运：船视宝小程序
- 新闻：财联社
- 市场：彭博社、路透社
- 分析：Grok AI

**品牌色**
- 主色：华泰蓝 1e40af
- 成功：16a34a
- 警告：f59e0b
- 危险：dc2626

- 地图高度调整为55vh
