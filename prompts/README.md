# 中东地缘跟踪网站 - 项目提示词总览

## 项目概述

这是一个专业的中东地缘政治冲突跟踪网站，主要用于跟踪美以伊战争期间的能源市场、航运通道和地缘风险。网站包含5个核心页面，采用统一的设计风格和专业金融信息网站的视觉规范。

## 品牌标识

- **品牌名称**：华泰固收 中东地缘跟踪
- **目标用户**：能源分析师、交易员、政策研究员、关注地缘风险的投资者
- **设计风格**：专业、严肃、数据驱动、信息密度高

## 统一设计系统

### 色彩体系
- **主色调**：#1e40af（深蓝）- 用于标题、重要强调
- **辅助色**：#3b82f6（亮蓝）- 用于链接、按钮高亮
- **背景色**：#f8fafc（浅灰蓝）- 页面背景
- **卡片背景**：#ffffff（白色）
- **边框色**：#e2e8f0（浅灰）
- **文字色**：#1e293b（深灰）正文，#64748b（中灰）次要文字
- **警示色**：#dc2626（红色）- 危险/紧急状态
- **警告色**：#f59e0b（橙色）- 警告信息
- **安全色**：#16a34a（绿色）- 正常/安全状态

### 导航栏（所有页面统一）
- 左侧：🌐 图标 + 【华泰固收】中东地缘跟踪（标题）
- 中间：5个导航按钮（药丸形状切换按钮组）
  - 🗺️ 原油图谱（链接到index.html）
  - 📰 每日简报（链接到briefing.html）
  - ⚡ 海峡跟踪（链接到tracking.html）
  - 🔴 实时新闻（链接到news.html）
  - 📈 Polymarket（链接到polymarket.html）
- 右侧：更新时间：2026年X月X日

### 响应式断点
- 桌面端：> 1024px
- 平板端：768px - 1024px
- 手机端：< 768px（导航按钮压缩，部分图标隐藏文字）

## 页面结构

| 页面 | 文件名 | 核心功能 | 主要数据源 |
|------|--------|----------|------------|
| 海湾原油图谱 | index.html | 交互式地图展示产油国分布、出口路线 | 手工维护+AI生成 |
| 美以伊冲突简报 | briefing.html | 每日战况、市场影响、投行观点 | AI自动生成 |
| 霍尔木兹跟踪 | tracking.html | 海峡通航数据、实时情报 | CSV+网络数据 |
| 实时新闻 | news.html | 财联社中东冲突专题新闻 | 爬虫自动抓取 |
| Polymarket | polymarket.html | 预测市场概率追踪 | Polymarket API |

## 技术栈

- **前端**：原生HTML + CSS + JavaScript
- **地图**：Leaflet.js (index.html, tracking.html)
- **图表**：Chart.js (tracking.html, polymarket.html)
- **CSS框架**：Tailwind CSS (polymarket.html)
- **字体**：Segoe UI / Inter

## 自动化更新

- **每20分钟**：新闻爬取(scrape_cls_final.py)、Polymarket更新(update_polymarket_html.py)
- **每12小时**：AI生成简报(update_briefing_grok.py)、地图内容更新(update_all_with_ai.py)
