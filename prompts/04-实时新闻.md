# 04-实时新闻 (news.html)

## 页面目标
创建一个新闻聚合页面，自动抓取财联社中东冲突专题的新闻，以时间倒序展示，支持分类筛选，确保用户第一时间获取权威信源的最新动态。

## 页面标题
【华泰固收】中东地缘跟踪 - 实时新闻

- **最大宽度**：1200px，居中显示
- **背景**：#f8fafc
- **内边距**：12px 16px

### 内容区域

#### 1. 来源标识头（Source Header）

- **左侧**：财联社Logo/标识 + "中东冲突专题"
- **右侧**："实时同步"标签
- **底部小字**：财联社是专业的财经新媒体，7x24小时报道中东局势

#### 2. 筛选栏（可选）
分类筛选按钮组：
- 全部
- 军事冲突 🔴
- 能源设施 🟠
- 航运通道 🔵

#### 3. 新闻列表（主内容区）
垂直排列的新闻卡片，每条新闻：

**新闻卡片结构**：
```html
<a href="原文链接" class="news-item" target="_blank">
  <div class="news-title">【标题】新闻标题...</div>
  <div class="news-source">

    <span>2026-03-19 08:47</span>
```

**卡片样式**：
- 白色背景
- 圆角8px
- 边框：1px solid #e2e8f0
- 内边距：16px
- 间距：12px（卡片之间）
- 悬停效果：背景变蓝灰(#eff6ff)，边框变蓝(#3b82f6)

**标题样式**：
- 字体：0.95rem
- 颜色：#1e40af（深蓝）
- 字重：600（半粗）
- 行前加【】标识，如【卡塔尔】、【油价】

**来源行样式**：
- 分类标签：小圆角药丸形状，根据分类不同颜色
- 时间：灰色小字

**摘要样式**：
- 字体：0.85rem
- 颜色：#475569
- 上边距：8px
- 顶部虚线边框分隔
- 最多3行，超出省略

"加载更多"按钮，点击加载下一页新闻

## 数据格式

新闻数据以JavaScript数组形式存储：

```javascript
const CLS_NEWS_DATA = [
  {
    id: "1",
    title: "【卡塔尔】拉斯拉凡天然气设施再次遭袭",
    summary: "记者当地时间19日凌晨获悉，卡塔尔拉斯拉凡天然气设施再次被导弹击中...",
    time: "2026-03-19 08:47",
    url: "https://www.cls.cn/detail/2317566",
    category: "energy" // energy/military/shipping/diplomacy
  }
  // ... 更多新闻
];
```

## 分类逻辑

根据标题关键词自动分类：

```javascript
function categorize(title) {
  const text = title.toLowerCase();
  if (text.match(/海峡|霍尔木兹|航运|油轮|船舶|港口|海运/)) 
    return 'shipping';
  if (text.match(/石油|原油|天然气|能源|油价|产量/)) 
    return 'energy';
  if (text.match(/外交|谈判|会谈|制裁|联合国|协议|停火/)) 
    return 'diplomacy';
  return 'military';
}
```

## 数据更新机制

### 爬虫脚本（scrape_cls_final.py）
- **目标URL**：https://www.cls.cn/subject/10986
- **技术**：Playwright + Chromium
- **频率**：每20分钟（GitHub Actions定时）
- **行为**：
  1. 打开财联社中东冲突专题页
  2. 点击"加载更多"按钮多次，加载全部新闻
  3. 提取每条新闻：标题、摘要、时间、链接
  4. 与现有新闻合并（去重）
  5. 按时间倒序排列
  6. 更新HTML中的JavaScript数组

### 增量更新
- 保留已有新闻
- 只添加新增新闻
- URL去重（同一链接只保留一条）
