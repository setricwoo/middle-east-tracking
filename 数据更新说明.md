# 数据更新说明

## 概述

`update_data_from_excel.py` 是一个独立的数据更新脚本，用于从 `全球市场.xlsx` 提取最新数据并更新到 `data-tracking.html` 页面。

**特点：**
- ✅ 只更新数据，不改变页面布局
- ✅ 不改变JavaScript逻辑
- ✅ 保留所有时间轴拖动功能
- ✅ 保留双Y轴显示

## 使用方法

### 方法一：双击运行（推荐）

直接双击 `更新数据.bat` 文件，脚本会自动执行并显示结果。

### 方法二：命令行运行

```bash
python update_data_from_excel.py
```

### 方法三：Python脚本调用

```python
from update_data_from_excel import update_html_data
update_html_data()
```

## 数据提取规则

### 商品价格数据
从Excel的**第一个sheet**提取：

| 分组 | 包含指标 |
|------|---------|
| 原油价格 | 布伦特原油、WTI原油 |
| 天然气价格 | NYMEX天然气、IPE英国天然气 |
| 石脑油与LPG | 石脑油、LPG |
| 甲醇与乙烯 | 甲醇、乙烯 |
| 铝与尿素 | LME铝、尿素 |
| 大豆与小麦 | CBOT大豆、CBOT小麦 |

**排除项：** 金融状况指数

### 流动性数据
从Excel的**第二个sheet**提取：

- 芝加哥联储全国金融状况指数
- 标准普尔500波动率指数(VIX)
- SOFR-IORB利差
- EURIBOR利率差
- 美债收益率利差(10Y-2Y)
- 企业债期权调整利差
- 中东汇率（新谢克尔、伊朗里亚尔等）

**排除项：** 
- 美元兑沙特里亚尔
- 空的全球股市隐含波动率

## 数据时间范围

- **保留数据：** 最近2年（730天）
- **默认显示：** 最近半年（180天），可通过时间轴拖动调整
- **时间顺序：** 正序（从早到晚）

## 更新后操作

1. 运行脚本后，会提示更新成功
2. 在浏览器中刷新 `data-tracking.html` 页面
3. 查看最新的商品价格和流动性数据

## 文件清单

| 文件 | 说明 |
|------|------|
| `全球市场.xlsx` | 数据源文件（需要保持此文件名） |
| `data-tracking.html` | 目标页面（会被更新） |
| `update_data_from_excel.py` | 数据更新脚本 |
| `更新数据.bat` | Windows批处理文件（双击运行） |

## 注意事项

1. 确保 `全球市场.xlsx` 文件与脚本在同一目录
2. Excel文件需要包含两个sheet：
   - Sheet 1: 商品价格数据
   - Sheet 2: 流动性数据
3. 不要修改Excel的表头结构（第1-5行的元数据格式）
4. 更新后需要刷新浏览器才能看到新数据

## 故障排除

**问题：脚本运行失败**
- 检查是否安装了 pandas: `pip install pandas openpyxl`
- 检查 `全球市场.xlsx` 是否存在

**问题：数据没有更新**
- 检查Excel文件是否已保存
- 检查浏览器缓存，强制刷新页面 (Ctrl+F5)

**问题：中文显示乱码**
- 这是正常的，不影响数据实际使用和显示
